# Vault map

- `projects/` — project context and plans
- `handoffs/` — human-readable active-work continuity
- `knowledge/decisions/` — reviewed decisions
- `knowledge/procedures/` — repeatable procedures
- `knowledge/research/` — research notes and landscapes
- `knowledge/troubleshooting/` — diagnosed failures and fixes
- `references/` — source references
- `deliverables/` — completed outputs
- `inbox/` — unclassified incoming material
- `.agent-kb/` — ignored agent runtime, tool checkout, and SQLite database
