# Vault contract

Reviewed Markdown is the human-facing authority for policy, navigation, projects, references, and deliverables. The vault-local `.agent-kb/kb.sqlite` is private agent runtime memory, not a replacement for Markdown or current source systems.

Agents search first, use handoffs for active continuity, create proposals when durable knowledge is uncertain, and promote only after review. Never store secrets or credentials. Keep durable claims attributable and update or supersede stale knowledge explicitly.
